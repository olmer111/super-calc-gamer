# 👑 Super-Calc Gamer

¡Bienvenido a la **Super-Calc Gamer**! No es una calculadora común; es una herramienta de productividad con un giro divertido para quienes no soportan el aburrimiento de las matemáticas tradicionales.

## 🚀 Características

### 🧮 Modo Calculadora
Una calculadora completa, elegante y con diseño neón para realizar todas tus operaciones matemáticas diarias. Rápida, precisa y visualmente atractiva.

### ⚡ Modo Juego: Súper Reflex
¿Crees que eres rápido con los números? Entra en el modo juego y pon a prueba tus reflejos:
- Te presentará sumas aleatorias.
- Debes escribir el resultado correcto lo más rápido posible.
- El marcador subirá con cada respuesta acertada.
- ¡Ideal para entrenar el cerebro mientras descansas!

## 🛠️ Tecnología Utilizada
- **HTML5**: Estructura semántica.
- **CSS3**: Diseño responsivo con variables modernas y estética "Dark Neon".
- **JavaScript (Vanilla)**: Lógica de cálculo y sistema de juego en tiempo real, sin dependencias externas para máxima velocidad.

## 📱 Cómo usarla en tu celular
1. Despliega el repositorio en **Vercel**.
2. Abre el link generado en tu navegador móvil.
3. ¡Disfruta de la potencia y los juegos en la palma de tu mano!
